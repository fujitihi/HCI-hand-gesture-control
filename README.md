# HCI Camera Control System
A project that enables PC control using camera, hand gestures, and audio input.

## Features
- Detects hand gestures using MediaPipe and a trained model
- Switches modes with claps
- Controls mouse and keyboard without touch
- Includes visualization overlay and keyboard interface

## Environment
- Python 3.10+
- OpenCV, MediaPipe, PyAudio, PyAutoGUI, NumPy, scikit-learn

## Run
```bash
python main.py
